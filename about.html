About Me

Hi, I’m Daniel Oluwafemi — a self-taught frontend web developer passionate about building beautiful, responsive, and interactive websites. I started my coding journey in 2024 because I wanted to turn ideas into real, usable designs that make people’s lives easier online.I started with HTML and CSS, learning how to create structured layouts and styled components. Soon after, I began exploring JavaScript to add functionality and interactivity,like toggling dark mode, building menus, and creating dynamic effects. I use VS Code as my code editor and GitHub to manage my projects. I've worked on several small projects like a bag collection product card, a birthday surprise website, and a mobile-friendly navigation bar. What I enjoy most is seeing my ideas come to life, pixel by pixel. Each project teaches me something new from structure to styling to scripting logic. My goal is to grow into a full-stack developer, learn backend development, and work with real clients to bring their ideas to life. I'm constantly learning and improving, and I believe every line of code brings me closer to who I want to be in tech.
